Habita プライバシーポリシー

とりのこ（以下、「当方」といいます。）は、提供するスケジュール通知サービス「Habita」（以下、「本サービス」といいます。）における、ユーザーの個人情報の取扱いについて、以下のとおりプライバシーポリシー（以下、「本ポリシー」といいます。）を定めます。

第1条（収集する情報）

当方は、本サービスの提供にあたり、以下の情報を取得および利用します。

Google アカウント連携により取得する情報

  • メールアドレスおよびプロフィール情報（認証のため）
  • Google カレンダーの予定情報（予定のタイトル、開始時刻、終了時刻、通知設定など、本サービスの通知に必要な範囲に限ります）

通知先プラットフォーム（LINE、Discord 等）から取得する情報

  • 各プラットフォームが発行するユーザー識別子（User ID）
  • その他、連携の設定に必要な公開プロフィール情報

本サービスの利用状況に関する情報

  • アクセスログ、エラーログ、Cookieなどの端末・ブラウザ情報

機密データのログ記録および参照の制限について

ユーザーのプライバシーを最大限保護するため、取得したメールアドレスやカレンダーの予定詳細（タイトルや内容など）は、システム内で厳重にフィルタリング処理を行います。
これらのデータはアクセスログやエラーログ等へは一切記録されず、平時において当方が直接参照できない状態を担保しております。
ただし、ユーザーご本人から不具合調査等の個別のお問い合わせをいただいた場合に限り、問題解決のための必要最小限の範囲において、特別に該当データを参照する場合があります。

第2条（利用目的）

当方は、取得した情報を以下の目的で利用します。
  • 本サービスへのログイン、ユーザー認証、および設定の保持のため
  • ユーザーが指定した Google カレンダーの予定時刻に基づき、連携済みの LINE または Discord アカウントへ通知を送信するため
  • 本サービスの提供、維持、保護、および改善のため
  • 本サービスに関するご案内、お問い合わせ等への対応のため
  • 利用規約に違反する行為に対する対応のため

第3条（個人情報の第三者提供）

当方は、次に掲げる場合を除いて、あらかじめユーザーの同意を得ることなく、第三者に個人情報を提供することはありません。ただし、個人情報保護法その他の法令で認められる場合を除きます。

Google アカウント連携により取得する情報

  • 人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難であるとき
  • 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、本人の同意を得ることが困難であるとき
  • 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合であって、本人の同意を得ることにより当該事務の遂行に支障を及ぼすおそれがあるとき

第4条（Google APIの利用に関する特記事項）

本サービスは、ユーザーのスケジュール管理機能を提供するため、Google APIから取得した情報を利用します。
本サービスが Google API から受け取った情報の使用および他のアプリへの転送は、限定的使用の要件（Limited Use requirements）を含む Google API Services User Data Policy に準拠します。
当方は、取得したカレンダー情報を本サービスの通知機能の提供のみに利用し、ユーザーの明示的な同意なく第三者に販売、譲渡、または広告目的で使用することはありません。

第5条（個人情報の開示・訂正・削除および連携の解除）

ユーザーは、本サービス上の設定画面、または各プラットフォーム（Google、LINE、Discord等）の連携解除画面より、いつでも本サービスとのデータ連携を解除することができます。
連携が解除された場合、当方は速やかに該当ユーザーの個人情報およびカレンダー情報をシステムから削除または匿名化します。

第6条（プライバシーポリシーの変更）

当方は、必要に応じて本ポリシーを変更することがあります。変更後のプライバシーポリシーは、本サービス上に掲載したときから効力を生じるものとします。ただし、利用目的の変更や、法令上ユーザーの同意が必要となるような変更を行う場合は、あらかじめユーザーの同意を取得します。
